Introduction

The following code sample demonstrates how to use the Microsoft Advertising SDK for Windows Phone to monetize your XNA based apps.

Building the Sample

To build and run this sample, you must have Visual Studio 2010 SP1 and the Microsoft Advertising SDK for Windows Phone installed.

  • Unzip the sample's solution file and supporting directory into your Visual Studio projects’ directory.
  • Open the solution file through Visual Studio 2010 or double click on the .sln file.
  • Press F5 to build the solution.

Description

This sample shows how to include advertisements in your XNA based Windows Phone apps. There is nothing fancy about the app. The app is simply an example that is used to show how you include advertisements using C# and XNA.

Source Code Files

  • XnaGameWithAd.zip - This code provides an example of how to use of the Microsoft Advertising SDK to show ads in your apps. It uses the XNA framework for Windows Phone.
