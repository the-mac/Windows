Introduction

The following code samples demonstrate how to use the Microsoft Advertising SDK for Windows Phone 7 to monetize your apps.

Building the Samples

To build and run this sample, you must have Visual Studio 2010 and the Microsoft Advertising SDK for Windows Phone installed.

  1. Unzip the sample's solution file and supporting directory into your Visual Studio projects' directory.
  2. Open the solution file through Visual Studio or double click on the .sln file.
  3. Press F5 to build the solution.


Description

This sample shows how to include advertisements in your Windows Phone 7 apps. There is nothing fancy about the app. The app is simply an example that shows how you can use the Advertising SDK to include ads in your apps.


Source Code Files

  • SilverlightAppWithAd.zip - This code provides an example of how to use of the Microsoft Advertising AdControl to show ads in your apps. It uses the Silverlight framework for Windows Phone. This sample was built in Visual Studio 2010 for Windows Phone 7. It will not work with Visual Studio 2012 and Windows Phone 8.
